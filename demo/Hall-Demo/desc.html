This demo is created from PMVS's included hall demo image set (source).

This demo is a reference project, it is mostly generated by external tools:

SIFT features are generated from Lowe's SIFT (source).

Feature matches are generated by WebMatcher (WebSFM's javascript implementation of kd-tree based ANN matcher).

Sparse reconstruction is recovered by Noah Snavely's Bundler (source).

Dense reconstruction is generated by Yasutaka Furukawa's PMVS (source).
