This demo is created from Leuven cityhall image set (source).

3D reconstruction of this demo is complete.
SIFT features are generated from WebSIFT (WebSFM's javascript implementation of SIFT);
Feature matches are generated by WebMatcher (WebSFM's javascript implementation of kd-tree based ANN matcher);
Sparse reconstruction is recovered by WebRegister (WebSFM's javascript implementation of Bundler and Sparse Bundle Adjustment), the initial pair of camera is still unstable;
Dense reconstruction is generated by Yasutaka Furukawa's PMVS (source), WebSFM's implementation of PMVS is still under construction;
